# Color Detection
 This project uses WebCam and tries to detect the color of the object in the center of the frame at a given instance

## Requirements :

* python 3.10
* opencv
* numpy

## Installing Required Libraries:

```pip install opencv-python```

```pip install numpy```

## Downloading the code 

``` git clone <repo link> ```
